# **App Name**: Code Ascent

## Core Features:

- Interactive Code Playground: An integrated, browser-based editor allowing users to write, execute, and debug code directly within educational exercises.
- Client-Side AI Feedback Tool: Provides instant, personalized feedback on user-written code, identifying errors and suggesting improvements by analyzing code patterns and logic directly in the browser.
- Dynamic Learning Path Adaption: An algorithm that automatically adjusts the user's learning sequence and presents tailored content based on their performance, strengths, and weaknesses identified from completed exercises.
- Visual Skill Mastery Tree: An interactive, visual representation of the user's progress and mastery across various programming topics, skill levels, and languages, reflecting achieved milestones.
- AI Practice Challenge Generator: Generates focused and customized coding challenges designed to target specific user weaknesses identified through their performance on previous exercises, utilizing client-side generation logic where feasible.

## Style Guidelines:

- Dark Mode Background: A deep, muted blue-grey (#111619) providing a strong contrast for luminous UI elements, hinting at technology and the night sky of data.
- Primary Color: A vibrant electric blue (#1F9EFF) used for key interactive elements, progress indicators, and to evoke a sense of digital clarity and focus.
- Accent Color: A striking neon green (#0DF20D) employed sparingly for highlights, successful actions, and special AI-driven notifications, providing high visibility and a futuristic edge.
- Headline Font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired aesthetic, suitable for section titles and prominent text.
- Body Font: 'Inter' (sans-serif) for high readability and neutrality in educational content, ensuring clarity for extended text passages.
- Code Font: 'Source Code Pro' (monospace) for crystal-clear display of code snippets and programming examples within the platform.
- Use minimalist, outline-style icons with a glowing effect when interactive, complementing the dark and neon aesthetic.
- Mobile-first, responsive design with clear card-based layouts and ample negative space to guide user focus, inspired by popular language learning applications.
- Subtle, smooth transitions for skill tree progression and module completion, using slight glows or pulsing effects on interactive elements.
